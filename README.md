# Requirements

* NodeJS

Install [Node](http://nodejs.org/)

* Flask

`<pip install flask>`

* React

`<npm install -g create-react-app>`

# Usage

**Step #1 :**

* Clone the Repo.

** Step #2 :**

* Install the Requirements

** Step #3 :**

* Fire up the terminal, enter the "flackbackend" folder, and execute this piece of snippet : `<python main.py>`
